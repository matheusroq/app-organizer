Simple ToDo web application.

### :gear: Tecnologias utilizadas:
* Node.js
* Express.js
* Sequelize
* Chart.js
### How to run
Install the packages
```
npm install
```
#### Back-end:
```
npm start
```
#### Front-end:
```
npm run dev
```
